# 豆包 Web App Bridge Skills — 中文介绍

## 这是什么

这是一个 **Claude Code Skill**，让 Claude Code 智能体能够直接操控浏览器里的豆包网页版，实现：

- 向豆包发送问题
- 读取豆包的回复
- 将完整对话保存为结构化文件（JSONL + Markdown）
- 导航回已保存的对话继续追问

整个过程不依赖任何 API Key，通过 Chrome DevTools Protocol（CDP）直接操作已登录的浏览器页面完成。

---

## 核心工作原理

### 消息发送

豆包使用字节跳动内部的 semi-design 组件库，输入框是一个 React 托管的 `<textarea>`。直接设置 `.value` 不会触发 React 的状态更新，send 按钮不会显示。

解决方案：通过 `element.__reactProps.onChange()` 直接调用 React 的内部事件处理器，绕过 DOM 事件系统，确保 React state 正确更新，send 按钮正常激活。

### 消息提取

豆包页面的 DOM 结构（已验证）：

| 元素 | 选择器 |
|---|---|
| 输入框 | `[data-testid="chat_input_input"]` |
| 发送按钮 | `[data-testid="chat_input_send_button"]` |
| 用户消息 | `[data-testid="send_message"]` |
| AI 回复 | `[data-testid="receive_message"]` |
| 消息正文 | `[data-testid="message_text_content"]` |
| 对话 URL | `https://www.doubao.com/chat/{纯数字ID}` |

不调用任何后端 API，直接读取渲染好的 DOM 内容，因此不依赖 Chat History 开关。

### navigate 注意事项

导航到已保存对话后，豆包页面需要约 4 秒完成渲染，才能正常识别输入框。操作规范：

```bash
python scripts/doubao_web_probe.py navigate --chat-id <chatId>
sleep 4
python scripts/doubao_web_probe.py probe   # 确认页面加载完毕
python scripts/doubao_web_probe.py ask --question "继续提问"
```

navigate 到具体对话 URL 需要已登录豆包账号。

## 对话存储格式

每个对话独立存储在 `~/.ai-bridge/doubao-bridge/conversations/` 下的子目录中：

```
{对话标题slug}--{chatId}/
  meta.json           ← 对话 ID、标题、URL、保存时间、标签
  conversation.jsonl  ← 每条消息一行 JSON，结构化存储
  conversation.md     ← LLM 友好的 Markdown 格式导出
```

存储在 skills 目录之外，升级或删除 skills 不会丢失对话记录。

### Markdown 格式示例

## 快速上手

### 1. 启动带 CDP 的 Chrome

```bash
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
  --remote-debugging-port=9222 \
  --user-data-dir=/tmp/doubao-cdp-profile
```

打开 `https://www.doubao.com/chat/` 并登录。

### 2. 安装依赖

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
```

### 3. 验证连接

```bash
python scripts/doubao_web_probe.py probe
python scripts/doubao_web_probe.py ask --question "回复：DOUBAO_BRIDGE_OK"
sleep 8
python scripts/doubao_web_probe.py read
```

预期：`sampleTail` 中包含 `DOUBAO_BRIDGE_OK` 的回复。

### 4. 保存对话

```bash
python scripts/doubao_conversation_store.py --export-md --project my-project
```

## Slash Command 用法

在 Claude Code 中使用 `/doubao` 命令：

| 命令 | 效果 |
|---|---|
| `/doubao start` | 开启路由模式，后续所有消息转发给豆包 |
| `/doubao end` | 关闭路由模式 |
| `/doubao +<消息>` | 单条消息转发给豆包 |
| `/doubao conversation list` | 列出所有已保存的对话 |
| `/doubao conversation <名称> +<消息>` | 导航到指定对话并发消息 |

`<名称>` 支持模糊匹配：关键词、缩写均可。

```
SKILL.md                       ← Claude Code skill 定义文件
```
